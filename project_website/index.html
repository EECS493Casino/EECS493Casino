Rain Man Casino Project

EECS 493 Winter 2015

Team members: Chris Elie, Martina Lugo, Troy Dietz, Addison Parker

The Project

For our class project we developed two casino card games, Poker and Blackjack. We wanted to emphasize a different feature in the two card games so we decided to change the implementation of the games by allowing the main player to cheat. Essentially, the main player can view the opponent's hand and know which cards are on the deck or will be dealt next. The game is single player to only give this cheating advantage to one person.

We will represent multiple views of the data, that is the hand that each of the players has plus the next cards coming up on the deck. To develop this project we used HTML and JavaScript as our main platforms.
